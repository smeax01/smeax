## Introduction

Félicitations ! Vous venez d'intégrer le support **Markdown** à votre projet Smeax. Markdown vous permet d'écrire du contenu riche de manière simple et intuitive, sans avoir à toucher au code HTML ou JavaScript.

### Pourquoi utiliser Markdown ?

1.  **Simplicité** : Pas besoin de balises HTML complexes.
2.  **Vitesse** : Idéal pour écrire des articles de blog ou de la documentation.
3.  **Flexibilité** : Vous pouvez mélanger Markdown et composants Astro si besoin.

### Exemple de mise en forme

Voici une liste des fonctionnalités supportées :

- **Gras** et _Italique_
- [Liens](https://astro.build)
- Listes à puces et numérotées
- Citations

> "Le code est comme l'humour. Quand vous devez l'expliquer, c'est qu'il est mauvais."

### Code Snippets

Vous pouvez également intégrer du code :

```javascript
function helloWorld() {
  console.log("Bonjour Smeax !");
}
```

N'hésitez pas à créer d'autres fichiers `.md` dans le dossier `src/pages/` pour ajouter de nouvelles pages !
